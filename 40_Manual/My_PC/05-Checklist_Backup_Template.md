## メンテナンスチェックリスト

### **Daily（毎日）**

* [ ] **ダウンロードファイルの削除**
  * 不要なインストーラーや一時ファイルを消去し、ストレージをクリーンに保つ。

* **ブラウザタブの整理**
  * [ ] 必要なものは「ブックマーク」へ。
  * [ ] 頻繁にアクセスするものは「RayCastのQuicklinks」へ登録。

* **Workフォルダの整理**
  * [ ] `/Users/ユーザ名/Documents/work` 内の一時データを適切な場所に移動。
  * [ ] SlackやSparkの一次情報をNotionやObsidianにストック。

### **Weekly（毎週）**

* [ ] **CCC（Carbon Copy Cloner）の実行**
  * 外付けHDD（500GB等）へ最新のデータをクローン。

* [ ] **VS Codeのコミット**
  * 各プロジェクトの変更漏れを確認し、GitHub等へPushして同期を完了させる。

### **Monthly（毎月）**

* [ ] **OSのアップデート**
  * macOS および Windows の最新セキュリティパッチを適用。

* [ ] **Brewfileの更新**
  * ターミナルで `brew bundle dump --force` を実行し、インストール済みリストを更新・バックアップ。

* [ ] **Eagleのデータ整理**
  * ライブラリ内の「重複アイテムをスキャン」し、不要なデータを削除。

## 緊急時の復旧チェックリスト (Disaster Recovery)

### **1. 最優先：認証情報の確保**

* [ ] **1Passwordへのログイン**
  * マスターパスワードとシークレットキーが手元にあるか確認。これがないと全ての復旧が止まるため。

### **2. 情報の復元**

* [ ] **Notionへのログイン**
  * 作業ログ、設定手順書、Wikiへのアクセスを確保。

* [ ] **GitHubへのログイン**
  * 最新のコード、設定ファイル（Dotfiles等）の取得。

### **3. OS・システム復元**

* [ ] **Macの復元 (Time Machine)**
  * 直近のバックアップポイントからシステム全体を復元。

* [ ] **Windowsの復元 (Macrium Reflect)**
  * 500GB HDDに保存したOSイメージからシステムを書き戻し。

### **4. 入力・操作環境の再構築**

* [ ] **Logi Options+ へのログイン**
  * マウス・キーボードのボタン割り当てプロファイルを同期。

* [ ] **Karabiner-Elements の設定適用**
  * バックアップしておいた `karabiner.json` を反映し、独自のキーマップを復旧。

---

**ヒント:** もしこれらを「Google Tasks」などのリマインダーアプリに登録して、自動的に通知が来るように設定したい場合はお手伝いできます。例えば「毎週日曜日の21時にCCC実行のリマインダーを作成して」と指示してください。
