
## 1. 基本のシステム操作

これらはすべてのアプリで共通して使える「必須級」のショートカットです。

|**アクション**|**ショートカット**|
|---|---|
|**コピー**|$\text{Cmd} + \text{C}$|
|**貼り付け**|$\text{Cmd} + \text{V}$|
|**カット**|$\text{Cmd} + \text{X}$|
|**全選択**|$\text{Cmd} + \text{A}$|
|**取り消し（元に戻す）**|$\text{Cmd} + \text{Z}$|
|**検索（Spotlight）**|$\text{Cmd} + \text{Space}$|
|**アプリの切り替え**|$\text{Cmd} + \text{Tab}$|
|**スクリーンショット（範囲指定）**|$\text{Cmd} + \text{Shift} + 4$|

---

## 2. Finder（ファイル操作）

ファイル管理をスムーズにするためのショートカットです。

- **新規Finderウィンドウを開く:** $\text{Cmd} + \text{N}$
    
- **クイックルック（中身をプレビュー）:** スペースキー（ファイル選択中）
    
- **ゴミ箱に入れる:** $\text{Cmd} + \text{Delete}$
    
- **ゴミ箱を空にする:** $\text{Cmd} + \text{Shift} + \text{Delete}$
    
- **ファイル名の変更:** $\text{Enter}$ （ファイル選択中）
    
- **上の階層に移動:** $\text{Cmd} + \uparrow$
    

---

## 3. ウィンドウとアプリの管理

画面が散らかるのを防ぎ、作業に集中するために役立ちます。

- **ウィンドウを閉じる:** $\text{Cmd} + \text{W}$
    
- **アプリを終了する:** $\text{Cmd} + \text{Q}$
    
- **アプリを隠す:** $\text{Cmd} + \text{H}$
    
- **強制終了ウィンドウ:** $\text{Cmd} + \text{Option} + \text{Esc}$
    
- **フルスクリーン切り替え:** $\text{Cmd} + \text{Ctrl} + \text{F}$
    

---

## 4. テキスト入力・ブラウザ操作

ドキュメント作成やネットサーフィンで多用します。

|**カテゴリ**|**アクション**|**ショートカット**|
|---|---|---|
|**テキスト**|**太字にする**|$\text{Cmd} + \text{B}$|
||**リンクを挿入**|$\text{Cmd} + \text{K}$|
|**ブラウザ**|**新しいタブを開く**|$\text{Cmd} + \text{T}$|
||**閉じたタブを再度開く**|$\text{Cmd} + \text{Shift} + \text{T}$|
||**アドレスバーを選択**|$\text{Cmd} + \text{L}$|
||**ページ更新**|$\text{Cmd} + \text{R}$|

## 5. 画面分割・整列の基本

最もよく使われる、画面を2分割や4分割にするショートカットです。

| **アクション**                  | **ショートカット**                                                             |     |
| -------------------------- | ----------------------------------------------------------------------- | --- |
| **画面の左半分に配置**              | $\text{Option} + \leftarrow$                                            |     |
| **画面の右半分に配置**              | $\text{Option} + \rightarrow$                                           |     |
| **画面の上半分に配置**              | $\text{Option} + \uparrow$                                              |     |
| **画面の下半分に配置**              | $\text{Option} + \downarrow$                                            |     |
| **全画面表示（Maximize）**        | $\text{Ctrl} + \text{Option} + \text{Return}$                           |     |
| **全画面表示（Almost Maximize）** | $\text{Option} + \text{Return}$                                         |     |
| **画面中央に配置（Center）**        | $\text{Ctrl} + \text{Option} + \text{C}$                                |     |
| **3分割の左 / 中 / 右:**         | $\text{Ctrl} + \text{Option} + \leftarrow$ / $\text{C}$ / $\rightarrow$ |     |
| **次のモニターへ**                | $\text{Option} + \text{N}$                                              |     |
| **前のモニターへ**                | $\text{Option} + \text{P}$                                              |     |

### その他の「ちょっとマニアック」な便利ショートカット

| **アクション**         | **ショートカット**                                             |
| ----------------- | ------------------------------------------------------- |
| **フォルダのパスを表示**    | $\text{Cmd} + \text{Shift} + \text{G}$                  |
| **ライブラリフォルダへ移動**  | Finderの「移動」メニューを開きながら $\text{Option}$ を保持               |
| **ファイルのパスをコピー**   | ファイルを選択して $\text{Option} + \text{Cmd} + \text{C}$       |
| **情報を見る（詳細）**     | $\text{Cmd} + \text{I}$                                 |
| **デスクトップを表示**     | $\text{Cmd} + \text{F3}$ （または $\text{Fn} + \text{F11}$） |
| **隠しファイルの表示・非表示** | **$\text{Cmd} + \text{Shift} + .$ （ピリオド）**              |

## 💡 知っておくと便利なヒント

- **記号の読み方:** * $\text{⌘} = \text{Command}$
    
    - $\text{⌥} = \text{Option (Alt)}$
        
    - $\text{⌃} = \text{Control}$
        
    - $\text{⇧} = \text{Shift}$
        
- **絵文字パレット:** $\text{Cmd} + \text{Ctrl} + \text{Space}$ でいつでも絵文字を呼び出せます。
