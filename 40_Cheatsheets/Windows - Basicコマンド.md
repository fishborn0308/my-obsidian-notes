
## 1. ファイル・テキスト操作

Macの $\text{Cmd}$ を $\text{Ctrl}$ に置き換えるだけで動くものが多いです。

|**アクション**|**ショートカット**|**Macでの対応**|
|---|---|---|
|**コピー**|$\text{Ctrl} + \text{C}$|$\text{Cmd} + \text{C}$|
|**貼り付け**|$\text{Ctrl} + \text{V}$|$\text{Cmd} + \text{V}$|
|**切り取り**|$\text{Ctrl} + \text{X}$|$\text{Cmd} + \text{X}$|
|**元に戻す**|$\text{Ctrl} + \text{Z}$|$\text{Cmd} + \text{Z}$|
|**上書き保存**|$\text{Ctrl} + \text{S}$|$\text{Cmd} + \text{S}$|
|**全選択**|$\text{Ctrl} + \text{A}$|$\text{Cmd} + \text{A}$|

---

## 2. システム・ウィンドウ管理

Windows独自の便利な管理機能です。

- **スタートメニューを開く:** $\text{Win}$
    
- **エクスプローラー（Finder相当）を開く:** $\text{Win} + \text{E}$
    
- **デスクトップを表示:** $\text{Win} + \text{D}$
    
- **検索を開く:** $\text{Win} + \text{S}$
    
- **アプリの切り替え:** $\text{Alt} + \text{Tab}$
    
- **ウィンドウを閉じる:** $\text{Alt} + \text{F4}$ （アプリの終了もこれ）
    
- **画面ロック:** $\text{Win} + \text{L}$ （離席時に必須）
    

---

## 3. ウィンドウ整列（スナップ機能）

Raycastを使わなくても、Windowsは標準で強力な整列機能を持っています。

- **左半分に配置:** $\text{Win} + \leftarrow$
    
- **右半分に配置:** $\text{Win} + \rightarrow$
    
- **最大化:** $\text{Win} + \uparrow$
    
- **最小化:** $\text{Win} + \downarrow$
    

---

## 4. 知ってると「通」な便利機能

- **クリップボード履歴:** **$\text{Win} + \text{V}$**
    
    - 過去にコピーしたものをリストから選んで貼り付けられます（超便利！）。
        
- **絵文字・記号パネル:** $\text{Win} + \text{.}$ （ピリオド）
    
- **スクリーンショット（範囲指定）:** $\text{Win} + \text{Shift} + \text{S}$
    
- **タスクマネージャー:** $\text{Ctrl} + \text{Shift} + \text{Esc}$
    
    - 動作が重いときにアプリを強制終了させるための最短経路です。

## 5. ブラウザ操作 (Chrome / Edge 共通)

タブの管理がマウスなしでできるようになると、ブラウジングのスピードが3倍変わります。

|**アクション**|**ショートカット**|**Macでの対応**|
|---|---|---|
|**新しいタブを開く**|**$\text{Ctrl} + \text{T}$**|$\text{Cmd} + \text{T}$|
|**今のタブを閉じる**|**$\text{Ctrl} + \text{W}$**|$\text{Cmd} + \text{W}$|
|**閉じたタブを復活させる**|**$\text{Ctrl} + \text{Shift} + \text{T}$**|$\text{Cmd} + \text{Shift} + \text{T}$|
|**タブを右/左に切り替え**|**$\text{Ctrl} + \text{Tab}$** / **$\text{Shift}$**|$\text{Ctrl} + \text{Tab}$|
|**検索バーにフォーカス**|**$\text{Ctrl} + \text{L}$** / **$\text{F6}$**|$\text{Cmd} + \text{L}$|
|**ページ内検索**|**$\text{Ctrl} + \text{F}$**|$\text{Cmd} + \text{F}$|

## 6. Excel (表計算)

Windows版Excelは、Mac版よりもショートカットの自由度が高く、これがWindowsを使い続ける理由という人も多いです。

- **セルの編集:** **$\text{F2}$** （Macは $\text{Ctrl} + \text{U}$）
    
    - _これを知らないと、毎回セルをダブルクリックすることになります。_
        
- **絶対参照の切り替え:** **$\text{F4}$** （$\text{\$A\$1}$ のように固定するやつです）
    
- **直前の操作を繰り返す:** **$\text{F4}$** または **$\text{Ctrl} + \text{Y}$**
    
    - _セルの色塗りや行の挿入を連続で行うときに神速です。_
        
- **フィルターをかける:** **$\text{Ctrl} + \text{Shift} + \text{L}$**
    
- **日付を入力:** **$\text{Ctrl} + \text{;}$** （セミコロン）
    
- **時刻を入力:** **$\text{Ctrl} + \text{:}$** （コロン）

## 7. 日本語入力・文字変換

Macの「かな」「英数」キーがない代わりに、Windowsでは一番上の「ファンクションキー」が威力を発揮します。

- **カタカナに変換:** **$\text{F7}$**
    
- **英字（半角）に変換:** **$\text{F10}$**
    
- **再変換:** 変換したい文字を選択して **「変換」キー**
    
- **カーソル移動（単語単位）:** **$\text{Ctrl} + \leftarrow / \rightarrow$**
    
    - _文章を一気に飛ばして移動できるので、修正が楽になります。_
        

---

## 8. プロの技：`Alt` キーの魔法

Windows最大の武器は、**`Alt` キーを1回叩く** ことです。

1. **`Alt` を押す:** 画面上部にアルファベットが表示されます。
    
2. **表示された文字を順に押す:** マウスなしでリボンの全機能にアクセスできます。
    
    - 例：`Alt` → `H` → `A` → `C` で「セルを中央揃え」など。

## 💡 Macユーザーがハマりやすいポイント

- **全角/半角の切り替え:** キーボード左上の `半角/全角` キー、または `Alt + ~`。
    
- **「名前の変更」:** ファイルを選択して **`F2`**。 (Macは `Enter` なので、Windowsで `Enter` を押すとファイルが開いてしまいます)
    
- **「ゴミ箱に入れず完全消去」:** `Shift + Delete`。
