# Command - Linux - nslookup - DNS サーバーに問い合わせて情報を取得する

## 概要

`nslookup` は、ドメイン名（FQDN）と IP アドレスの対応関係を調査するためのツールです。特定の DNS サーバーを指定して問い合わせたり、A レコード、MX レコード、TXT レコードといった特定のレコードの種類を指定して情報を取得したりすることができます。

## 🛠 `nslookup` vs `dig`

| 項目 | `nslookup` | `dig` (Domain Information Groper) |
| :--- | :--- | :--- |
| **特徴** | **対話モード** があり、初心者にも扱いやすい。 | 出力が詳細で、スクリプトでのパースが容易。 |
| **普及率** | Windows にも標準搭載。 | Linux/UNIX 系のデファクトスタンダード。 |
| **推奨度** | クイックな確認用。 | **2026年の標準（詳細調査用）。** |

## 主要な使い方

`nslookup` には「非対話モード」と「対話モード」の 2 つがあります。

### 1. 非対話モード (単発実行)

```bash
# 基本: ドメインから IP を調べる (正引き)
nslookup google.com

# 特定の DNS サーバー (例: 8.8.8.8) を使って調べる
nslookup google.com 8.8.8.8
````

### 2. 対話モード (深掘り)

コマンド名だけを打つとプロンプト `>` が表示され、連続して設定変更や問い合わせが可能です。

Bash

```
nslookup
> set type=mx        # メールサーバー(MXレコード)を調べる設定
> yahoo.co.jp       # 対象ドメインを入力
> set type=txt       # SPF設定などのTXTレコードを調べる設定
> google.com
> exit               # 終了
```

## 業務シナリオ別活用法

### 1. インフラ構築・運用視点

- **タスク**: メールが届かない原因の調査。
- **解説**: `set type=mx` で、宛先ドメインのメールサーバーが正しく設定されているか、また優先順位はどうなっているかを確認します。
- **例**:

    Bash

    ```
    nslookup -type=mx example.com
    ```

### 2. ブルーチーム視点

- **タスク**: フィッシングサイトのインフラ調査。
- **解説**: 不審なドメインの IP アドレスを特定し、その IP がどのプロバイダ（ホスティング）に属しているかを逆引き (`nslookup <IP>`) で確認して、テイクダウン（閉鎖）要請の材料にします。

### 3. レッドチーム視点

- **タスク**: ターゲット組織の公開資産（アタックサーフェス）の偵察。
- **解説**: TXT レコードには Google Workspace や Office 365 の認証情報、SPF レコード（メール送信元制限）が含まれており、そこからターゲットが利用しているクラウドサービスを推測できます。

## 注意点・補足

- **非推奨（Deprecated）通知**: 一時期、`nslookup` は「非推奨なので `dig` を使え」と言われていましたが、あまりに広く使われているため、現在は開発が継続され、モダンな環境でも標準的に使われています。
- **権限のない回答 (Non-authoritative answer)**: これは「そのドメインの公式サーバーではなく、キャッシュサーバー（プロバイダの DNS 等）が答えを返した」ことを意味します。内容が古い可能性があるため、正確を期すなら公式サーバーへ直接問い合わせましょう。

## セキュリティに関する考慮事項

### 脆弱性とリスク

- **情報の露出 (Information Disclosure)**: ⭐️ **重要**。

    かつての DNS サーバーでは「ゾーン転送 (`ls -d <domain>`)」という、ドメイン内の全レコードを一括取得するコマンドが `nslookup` 内で使えましたが、現在はセキュリティ上の理由でほとんどのサーバーがこれを拒否します。これが成功してしまうサーバーは設定不備です。

### 対応策・緩和策

- **DNS レコードの最小化**: 公開する必要のないサブドメインや、古いデバッグ用の TXT レコードなどを放置しないように管理を徹底します。
