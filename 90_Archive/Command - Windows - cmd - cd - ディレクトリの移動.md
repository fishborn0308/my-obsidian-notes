# Command - Windows - cmd - cd - ディレクトリの移動

## 概要

`cd` (Change Directory) または `chdir` は、現在の作業ディレクトリ（カレントディレクトリ）を表示または変更するためのコマンドです。Windows ではドライブという概念があるため、ドライブを跨ぐ移動には特定のスイッチが必要になります。

(出自: `Windows 標準搭載 - cmd.exe 内蔵コマンド`)

## 類似コマンドと差異

| 類似コマンド | 差異 / 使い分けのポイント | 推奨 (あれば) |
| :--- | :--- | :--- |
| `pushd` / `popd` | 移動先をスタックに保存し、元のディレクトリに簡単に戻れる。ネットワークパス (UNC) にも直接移動可能。 | 複雑な往復時 |
| `Set-Location` (PS) | PowerShell版。エイリアスとして `cd` が使えるが、中身は別物。 | PS環境 |

## よく連携されるコマンド

### シナリオ例: 書き込み可能ディレクトリへの迅速な退避 (レッドチーム視点)

* **目的**: 権限の低いユーザーで侵入した際、ファイルをアップロード・実行できるディレクトリへ移動する。
* **連携コマンド**: `cd`, `dir`
* **解説**: Windows において `C:\Windows\Temp` や `C:\Users\Public` は低権限でも書き込み可能な場合が多く、攻撃の拠点となります。
* **コマンド例**:
    ```cmd
    REM ドライブを跨いで Temp ディレクトリへ移動
    cd /d C:\Windows\Temp
    ```

## スイッチ/オプション説明

| スイッチ/オプション | 説明 |
| :--- | :--- |
| **引数なし** | 現在のドライブ名とディレクトリ名を表示する。 |
| ⭐ `/D` | 現在のドライブを変更した上で、ディレクトリも変更する（例: CドライブからDドライブへ移動）。 |
| `..` | 1つ上の親ディレクトリに移動する。 |
| `\` | ドライブのルートディレクトリに移動する。 |

## よく使われる組み合わせと業務シナリオ

### 1. インフラ構築・運用視点

* **タスク**: ログファイルを格納している別ドライブのフォルダへ移動して作業する。
* **例**:
    ```cmd
    REM Dドライブの Logs フォルダへ一気に移動
    cd /d D:\AppServer\Logs
    ```

### 2. レッドチーム視点

* **タスク**: ターゲット端末内でのディレクトリ列挙 (Enumeration)。
* **例**:
    ```cmd
    REM ユーザーのデスクトップへ移動してファイルを物色
    cd /d %USERPROFILE%\Desktop
    ```

## エラーメッセージとトラブルシューティング

1.  **エラーメッセージ例 1**: `The system cannot find the path specified.`
    * **考えられる原因**: パスのスペルミス、またはスペースを含むパスを引用符 `"` で囲んでいない。
    * **解決策**: `cd "C:\Program Files"` のように引用符を使用する。

## セキュリティに関する考慮事項

* **情報収集**: どのディレクトリに移動可能か（アクセス権があるか）を調べること自体が、内部偵察の重要なプロセスになります。
