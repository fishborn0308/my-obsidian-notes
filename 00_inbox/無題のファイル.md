## 2. ツール管理とPython環境 (`pipx`)

システムを汚さず、どこからでもツールを呼び出せるようにします。

* **pipxのセットアップ:**
```bash
sudo apt install -y pipx && pipx ensurepath

```


* **主要ツールのインストール:**
* `pipx install git+https://github.com/Tib3rius/AutoRecon.git` (自動列挙)
* `pipx install impacket` (ネットワーク攻撃)
* `pipx install netexec` (AD列挙・攻撃)
* `pipx install updog` (アップロード対応HTTPサーバ)



---

## 3. ターミナルと操作効率の強化 (`.zshrc`)

24時間の試験で疲弊しないための「爆速」設定です。

* **必須パッケージ:** `fzf`, `zoxide`, `zsh-autosuggestions`, `zsh-syntax-highlighting`, `tmux`, `rlwrap`
* **`.zshrc` への追記内容:**
```bash
# 曖昧検索(fzf)と移動(zoxide)の有効化
source /usr/share/doc/fzf/examples/key-bindings.zsh
eval "$(zoxide init zsh --cmd z)"

# 補完とハイライト
source /usr/share/zsh-autosuggestions/zsh-autosuggestions.zsh
source /usr/share/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh

# 便利なエイリアス
alias pserv="python3 -m http.server 80"
alias udot="updog -p 80"
alias mterm="sudo thunar /opt"

```



---

## 4. 証拠保全とノート作成

* **スクリーンショット:** `Flameshot` をインストールし、`PrintScreen` キーに `flameshot gui` を割り当て。
* **ノート:** `Obsidian` 等を使用し、階層構造でメモを管理。
* **配置ルール:**
* `/opt`: インストールが必要な大型ツール（root権限管理）
* `~/oscp/scripts`: 頻繁にターゲットへ送るスクリプト（一般ユーザー権限）

## 5. 最終チェックリスト（構築完了後の儀式）

### □ システム・操作性

* [ ] ホストOSとのコピペ・ファイルドラッグが可能か？
* [ ] `z <名前の一部>` でディレクトリジャンプができるか？
* [ ] `Ctrl + R` でコマンド履歴の曖昧検索ができるか？
* [ ] `PrintScreen` で Flameshot が起動し、矢印などの注釈が入れられるか？

### □ ツール・ネットワーク

* [ ] `impacket-psexec` 等がどのディレクトリからでも起動するか？
* [ ] `updog -p 80` で起動し、別端末からファイルのアップロードができるか？
* [ ] `rockyou.txt` が解凍されているか？ (`/usr/share/wordlists/`)

### □ 保険

* [ ] **【最重要】すべての設定が終わった状態で「クリーンなスナップショット」を撮影したか？**
