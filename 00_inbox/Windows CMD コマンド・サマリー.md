# Windows CMD コマンド・マスターサマリー (2026/02/22 時点)

## 1. 偵察・列挙 (Reconnaissance & Enumeration)

ターゲットの全容を解明するための、ペンテストにおける最重要カテゴリです。

|**コマンド**|**主な用途**|**ポイント**|
|---|---|---|
|**`dir`**|ファイル/フォルダの列挙|`/ah` で隠しファイル、`/s` で再帰検索。|
|**`find`**|文字列検索（簡易）|パイプでの絞り込み、行数カウント（`/c`）。|
|**`findstr`**|高度な文字列検索|**正規表現**対応。`/s` でサブフォルダまで一括検索。|
|**`driverquery`**|ドライバ情報の表示|`/v` でパス特定、`/si` で署名確認。LPE の調査に。|
|**`nslookup`**|DNS クエリ|ドメイン構造やゾーン転送の確認。|
|**`net view`**|共有リソースの列挙|同一ドメイン内のホストや共有フォルダの特定。|
|**`net session`**|アクティブセッション確認|ログイン中ユーザー（管理者等）の特定。|

## 2. 権限・アカウント管理 (Identity & Access)

ユーザー情報やグループ権限を操作し、横展開（Lateral Movement）の準備をします。

|**コマンド**|**主な用途**|**ポイント**|
|---|---|---|
|**`net user`**|ユーザー情報の操作|パスワード変更、ドメインユーザーの列挙。|
|**`net group`**|ドメイングループ管理|Domain Admins などの特権グループの特定。|
|**`net localgroup`**|ローカルグループ管理|`Administrators` への追加など、権限昇格の確認。|
|**`net accounts`**|パスワードポリシー確認|ロックアウト設定の確認。|

## 3. ログ・診断・システム修復 (Log, Diagnosis & Repair)

ブルーチームとしての調査や、壊れたシステムの復旧、高度な構成変更に使用します。

|**コマンド**|**主な用途**|**ポイント**|
|---|---|---|
|**`eventvwr`**|イベントログ閲覧 (GUI)|視覚的なログ確認の起点。|
|**`wevtutil`**|イベントログ操作 (CLI)|**ログの全消去** (`cl`) や特定 ID の抽出。|
|**`dism`**|システムイメージの修復|OS 機能の有効化 (`Enable-Feature`) や破損修復。|
|**`chkdsk`**|ディスクエラーチェック|ファイルシステムの整合性確認。|
|**`fc` / `comp`**|ファイル比較|設定の差異やバイナリ改ざんの特定。|

## 4. 自動化・シェル環境制御 (Automation & Environment)

単一のコマンドを「武器」や「ツール」に昇華させるための制御コマンドです。

|**コマンド**|**主な用途**|**ポイント**|
|---|---|---|
|**`for`**|ループ処理|**Ping Sweep** や一括ファイル処理の自動化。|
|**`echo`**|メッセージ出力/ファイル作成|ペイロードの動的生成 (`>>`)。|
|**`setlocal` / `endlocal`**|変数スコープの限定|スクリプト内での環境変数汚染を防止。|
|**`doskey`**|履歴/マクロ管理|履歴消去 (`/reinstall`) やエイリアス作成。|
|**`exit`**|シェルの終了|終了コード (`ERRORLEVEL`) の返却。|
|**`call`**|別バッチの呼び出し|スクリプトのモジュール化。|

## 5. ファイル・ストレージ操作 (File & Storage)

データの秘匿、圧縮、展開、そして破壊に関わる操作です。

|**コマンド**|**主な用途**|**ポイント**|
|---|---|---|
|**`diskpart`**|ディスク管理|隠しパーティションの操作、VHD のマウント。|
|**`cipher`**|暗号化/空き領域消去|データの物理的抹消。|
|**`compact`**|NTFS 圧縮|持ち出しデータのサイズ削減。|
|**`expand`**|配布用ファイル展開|CAB ファイル等からのバイナリ抽出。|
|**`del` / `erase`**|ファイル削除|痕跡消去。ごみ箱を経由しない即時削除。|
|**`net use`**|リモート共有のマウント|認証情報を使用した他端末への接続。|
|**`net share`**|共有フォルダの作成|データ持ち出し用フォルダの公開。|

### 次のステップ

アルファベット順を再開すると、次は **`fsutil`** です。

> **`fsutil`** は、ファイルシステム（NTFS）のメタデータを直接操作できる非常に強力なツールです。
> 
> 例えば、ファイルの操作履歴を記録する「USN ジャーナル」の調査（フォレンジック）や、巨大な「スパースファイル（ダミーファイル）」の瞬時作成など、OS の深部に触れる機能が満載です。
