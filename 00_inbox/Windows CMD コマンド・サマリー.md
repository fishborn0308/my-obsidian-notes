## Windows CMD コマンド・サマリー

現在までに **24個**（`net` サブコマンド含む）のドキュメントが完了しています。

### 1. ネットワーク偵察・リソース探索 (Reconnaissance)

ターゲットの構造や共有リソースを特定するための、攻撃・防御双方の最重要カテゴリです。

|**コマンド**|**概要**|**主な用途 (Red/Blue)**|
|---|---|---|
|**`nslookup`**|DNS 情報のクエリ|ドメイン構造の把握、ゾーン転送の試行|
|**`net view`**|ネットワーク共有の列挙|ネットワーク内のホスト特定、公開共有の探索|
|**`net session`**|リモート接続の表示|管理者がログインしている端末（HVT）の特定|
|**`dir`**|ディレクトリ表示|隠しファイルの発見、機密情報（config等）の探索|

### 2. アカウント・権限管理 (Account & Privilege)

Active Directory 環境やローカル端末での特権奪取、列挙に関わるコマンド群です。

|**コマンド**|**概要**|**主な用途 (Red/Blue)**|
|---|---|---|
|**`net user`**|ユーザー管理|ドメインユーザーの列挙、パスワード変更|
|**`net group`**|ドメイングループ管理|ドメイン管理者の特定（Domain Admins）|
|**`net localgroup`**|ローカルグループ管理|ローカル管理者の列挙、特権グループへの追加|
|**`net accounts`**|パスワードポリシー|ロックアウト閾値の確認（スプレー攻撃の準備）|

### 3. 横展開・リソース操作 (Lateral Movement)

特定したリソースへ接続し、データを操作・移動させるためのコマンド群です。

|**コマンド**|**概要**|**主な用途 (Red/Blue)**|
|---|---|---|
|**`net use`**|リモートリソース接続|共有ドライブのマウント、認証セッションの確立|
|**`net share`**|ローカル共有管理|データの持ち出し用共有（ドロップゾーン）の作成|
|**`net start/stop`**|サービスの開始・停止|防御機能（AV/EDR）の停止、永続化の確認|

### 4. ファイル・ディスク整合性 (File & Disk Integrity)

データの秘匿、改ざん検知、証拠隠滅に関わる高度な管理コマンドです。

|**コマンド**|**概要**|**主な用途 (Red/Blue)**|
|---|---|---|
|**`cipher`**|EFS暗号化・空き領域消去|データの完全消去（証拠隠滅）、暗号化の適用|
|**`comp`**|ファイル比較|バイナリ改ざん検知、バージョンの特定|
|**`compact`**|NTFS 圧縮|持ち出しデータの圧縮、ストレージの節約|
|**`chkdsk`**|ディスク修復|ファイルシステムの整合性確認、DoS への悪用|
|**`del / erase`**|ファイル削除|攻撃痕跡の消去、自己削除スクリプトの作成|

### 5. シェル環境・システム設定 (Environment & Config)

操作環境を整え、設定を操作するための基礎コマンドです。

|**コマンド**|**概要**|**主な用途 (Red/Blue)**|
|---|---|---|
|**`cmd`**|インタープリター開始|子プロセスの生成、難読化コマンドの実行|
|**`call`**|バッチ呼び出し|スクリプトのモジュール化、制御フローの構築|
|**`chcp`**|文字コード変更|UTF-8 環境への対応、地域設定の推測|
|**`cls` / `color`**|画面消去 / 色変更|画面上の痕跡隠蔽、操作環境の識別|
|**`control`**|設定ダイアログ起動|制限されたシェルの回避（KIOSK突破等）|
|**`date`**|システム日付管理|タイムスタンプ改ざん（Timestomping）|
