# Cheatsheet - Obsidian Regular Expression

# Obsidian 正規表現チートシート

Obsidian内での検索（`Ctrl/Cmd + F`）や、コミュニティプラグインでの高度な置換・抽出に役立つ正規表現のまとめです。

---

## 1. 基本の特殊文字（メタ文字）

|**文字**|**説明**|**例**|
|---|---|---|
|`.`|任意の1文字（改行を除く）|`a.c` → `abc`, `a-c`, `a1c`|
|`\`|次の文字をエスケープ（特殊文字をただの文字として扱う）|`\.` → `.` という文字そのもの|
|`^`|行の先頭|`^#` → 行頭にある見出しの `#`|
|`$`|行の末尾|`です。$` → 行末が「です。」で終わる部分|
|`\|`|OR（または）|`Apple\|Orange` → `Apple` か `Orange`|

---

## 2. 文字クラス（集合）

|**文字**|**説明**|**例**|
|---|---|---|
|`[abc]`|a, b, c のいずれか1文字|`[012]` → 0か1か2|
|`[^abc]`|a, b, c 以外の1文字|`[^0-9]` → 数字以外|
|`[a-z]`|aからzまでの範囲の1文字|`[a-zA-Z]` → アルファベット全般|
|`\d`|数字 `[0-9]` と同じ|`\d\d` → 2桁の数字|
|`\D`|数字以外||
|`\w`|英数字とアンダースコア `[a-zA-Z0-9_]`||
|`\s`|空白文字（スペース、タブ、改行）||
|`\S`|空白文字以外||

---

## 3. 量指定子（繰り返し）

|**文字**|**説明**|**例**|
|---|---|---|
|`*`|0回以上の繰り返し（最大）|`ab*` → `a`, `ab`, `abbb`|
|`+`|1回以上の繰り返し（最大）|`ab+` → `ab`, `abbb`（`a`単体は不可）|
|`?`|0回または1回（あってもなくても良い）|`https?` → `http`, `https`|
|`{n}`|ちょうど n 回|`\d{4}` → 西暦などの4桁数字|
|`{n,}`|n 回以上|`\d{3,}` → 3桁以上の数字|
|`{n,m}`|n 回以上 m 回以下|`\d{2,4}` → 2〜4桁の数字|
|`*?` / `+?`|**最短一致**（最小限の範囲で一致させる）|`\[.+?\]` → `[[WikiLink]]` の抽出に必須|

---

## 4. グループ化と参照

|**文字**|**説明**|**用途**|
|---|---|---|
|`( )`|グループ化|一つにまとめて量指定子を適用する|
|`$1`, `$2`|置換時の参照|検索したグループの内容を置換先で再利用する|

> [!TIP] 置換の例
>
> **検索:** `(\d{4})-(\d{2})-(\d{2})` (例: 2023-10-25)
>
> **置換:** `$1年$2月$3日`
>
> **結果:** 2023年10月25日

---

## 5. Obsidianでよく使う実践パターン

### 内部リンク（WikiLink）を探す

`\[\[.+?\]\]`

- `[[ノート名]]` や `[[ノート名|表示名]]` にマッチします。

### タグを探す

`#[^\s#]+`

- `#Tag` にマッチします（スペースや別の#が現れるまで）。

### 特定のプロパティ（YAML）を抽出

`^status: .+$`

- `status:` で始まる行を丸ごと抽出します。

### チェックボックス（未完了）を探す

`^\s*- \[ \] .+$`

- 行頭のインデントを考慮した未完了タスク。

### 箇条書きを番号付きリストに変換（置換）

**検索:** `^\s*-\s+(.+)$`

**置換:** `1. $1`
