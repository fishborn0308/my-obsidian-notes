# Cheatsheet - Obsidian Shortcut

# Obsidian ショートカットキー・チートシート

> [!NOTE] 表記について
>
> - `Cmd` = Mac / `Ctrl` = Windows・Linux
> 		
> - `Opt` = Mac / `Alt` = Windows・Linux
> 		

## 1. 基本操作（ファイル・UI）

|**ショートカット**|**機能**|
|---|---|
|`Cmd/Ctrl + N`|**新規ノート**を作成|
|`Cmd/Ctrl + O`|**クイック切り替え**（ノートを検索して開く）|
|`Cmd/Ctrl + W`|現在のタブを閉じる|
|`Cmd/Ctrl + P`|**コマンドパレット**（すべての機能を検索・実行）|
|`Cmd/Ctrl + ,`|設定を開く|
|`Cmd/Ctrl + Alt + ←/→`|前/次の履歴へ戻る・進む|
|`Cmd/Ctrl + Shift + F`|**グローバル検索**（保管庫全体から検索）|

---

## 2. エディタ操作（編集・書式）

|**ショートカット**|**機能**|
|---|---|
|`Cmd/Ctrl + Enter`|**チェックボックス**の作成 / 状態切り替え|
|`Cmd/Ctrl + B`|**太字**（Bold）|
|`Cmd/Ctrl + I`|_斜体_（Italic）|
|`Cmd/Ctrl + K`|外部リンクを作成 `[ ]( )`|
|`Cmd/Ctrl + /`|現在の行をコメントアウト `%% %%`|
|`Cmd/Ctrl + D`|現在の行を削除（設定で割り当てが必要な場合あり）|
|`Alt + ↑ / ↓`|現在の行を**上下に移動**|

---

## 3. 表示モード・ビュー

|**ショートカット**|**機能**|
|---|---|
|`Cmd/Ctrl + E`|**閲覧モード** と **ライブプレビュー** の切り替え|
|`Cmd/Ctrl + Click`|リンクを**新しいタブ**で開く|
|`Cmd/Ctrl + Alt + Enter`|リンクを**新しいウィンドウ**で開く|
|`Cmd/Ctrl + \`|サイドバーの表示 / 非表示切り替え|

---

## 4. インフラ・技術学習に便利な操作

### コードブロックの挿入

デフォルトではショートカットがありませんが、**「コマンドパレット」**から以下の手順で設定すると、CTFのログやコードを貼る際に非常に便利です。

1. `Cmd/Ctrl + ,` で設定を開く
2. 「ホットキー」を選択
3. 「コードブロックを挿入」を検索し、`Cmd/Ctrl + Shift + C` などに割り当て

### プロパティの編集

|**ショートカット**|**機能**|
|---|---|
|`Cmd/Ctrl + Alt + P`|プロパティ（YAML）にフォーカスする|

---

## 5. 【推奨】自分専用のカスタマイズ

インフラ構築やセキュリティ学習のメモを効率化するために、以下のホットキー設定も検討してみてください。

- **テンプレートの挿入**: `Alt + T` など（OSCPの学習用テンプレートなどを即座に呼び出す）
- **本日のデイリーノートを開く**: 毎日一からログを取る場合に便利です。
