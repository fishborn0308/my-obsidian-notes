# Ollama 導入・運用ガイド (Windows / Minisforum M1 Pro-285H 向け)

## 1. インストール手順

1. **公式サイトからダウンロード:** [ollama.com](https://ollama.com/) にアクセスし、Windows版をダウンロードします。
2. **インストーラーの実行:** `.exe` ファイルを実行し、案内に従ってインストールを完了させます。
3. **起動確認:** 画面右下のタスクトレイに「ラマ（🦙）」のアイコンが表示されていれば、バックグラウンドで動作しています。

## 2. モデルのダウンロードと管理

ターミナル（PowerShell または コマンドプロンプト）を開き、以下のコマンドを使用します。

### 主要なモデルの取得

* **汎用（軽量・高速）:** `ollama run llama3.1` (8Bモデル)
* **日本語特化:** `ollama run gemma2` (9Bモデル)
* **最高性能（低速）:** `ollama run llama3.3` (70Bモデル)

### 管理コマンド一覧

| コマンド | 内容 |
| --- | --- |
| `ollama list` | インストール済みモデルの一覧表示 |
| `ollama rm [モデル名]` | 不要なモデルの削除 |
| `ollama pull [モデル名]` | 実行せずにダウンロードだけ行う |

---

## 3. 実践的なプロンプトの投げ方

ターミナルでの対話（`>>>` 表示時）における操作方法です。

### 基本操作

* **一行入力:** そのまま文章を打って Enter。
* **複数行入力:** `"""` で囲む。
```text
>>> """
ここに長い文章や
ソースコードを貼り付ける
"""

```


* **終了:** `/bye` と入力。

### システムプロンプト（役割設定）の例

モデルの振る舞いを固定したい場合、最初にこう伝えます。

> `あなたはプロのITコンサルタントです。専門用語を避け、中学生にもわかるように解説してください。`

---

## 4. パフォーマンス最適化（Minisforum特有のポイント）

Core Ultra 9 285Hの性能を引き出すためのチェックリストです。

* **GPUアクセラレーション:** タスクマネージャーの「パフォーマンス」タブで、推論中に **Intel Arc Graphics (GPU)** が稼働しているか確認してください。
* **メモリ割り当て:** 64GBあるため、70Bモデルを動かす際は他の重いアプリ（ブラウザのタブ大量開きなど）を閉じると、スワップが発生せず安定します。
* **速度の目安:**
* **8B/9Bモデル:** リアルタイム（人間が読むより速い）
* **70Bモデル:** 1文字ずつゆっくり（熟考型）



---

## 5. 推奨エコシステム（GUI化）

黒い画面を卒業し、快適に使うためのツールです。

* **Page Assist (ブラウザ拡張):** ブラウザのサイドバーでChatGPTのようにOllamaを使えます。
* **Chatbox AI:** Windowsアプリとして動作。UIが綺麗で、プロンプトのテンプレート保存が可能です。
